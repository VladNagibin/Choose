Choose is an instrument to create a votes

  • Num to choose:
    • Quantity of cards that user need to choose
  • Settings button:
    • Open and close settings panel. Available only to vote's creater
  • Cards:
    • Data array that was uploaded in vote creation. Card is unavailable if maximum of users choose it.You can hide unavailable cards or remove the limit
  • Data fields:
    • Set when creating a vote. Can be changed in settings panel
